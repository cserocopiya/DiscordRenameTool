# Discord Nickname Updater (Bulk Rename Bot)

Скрипт для массового изменения никнеймов пользователей на Discord-сервере на основе текстового списка. Скрипт сопоставляет Discord ID участников с желаемыми игровыми никнеймами, переименовывает их и выводит статистику, включая список игроков, которых нет на сервере.

Реализован на двух языках: **Node.js (JavaScript)** и **Python**. Вы можете использовать любой удобный вам вариант.

## Функционал
- Чтение списка формата `<Игровой_Ник> <Discord_ID>` из текстового файла.
- Пропуск пользователей, у которых никнейм уже совпадает с желаемым (для экономии лимитов Discord API).
- Задержка между запросами (1 секунда) для защиты от жесткого рейт-лимита.
- Вывод итоговой статистики в конце работы.
- Отдельный список пользователей, которых не удалось найти на сервере.

## Подготовка бота в Discord Developer Portal
Для работы скрипта требуется Discord Bot:
1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications).
2. Создайте приложение и добавьте бота в разделе **Bot**.
3. Прокрутите страницу бота вниз до подраздела **Privileged Gateway Intents** и **обязательно включите** переключатель **Server Members Intent** (без него бот не сможет получить список участников).
4. Перейдите во вкладку **OAuth2 URL Generator**, выберите область `bot` и права `Manage Nicknames` (Управлять никнеймами), скопируйте ссылку и пригласите бота на ваш сервер.
5. На самом сервере в настройках ролей переместите роль вашего бота **выше ролей участников**, которых нужно переименовать (иначе Discord выдаст ошибку отсутствия прав).

## Настройка проекта
1. Склонируйте репозиторий.
2. Скопируйте файл `.env.example` и переименуйте его в `.env`:
   ```bash
   cp .env.example .env
   ```
3. Заполните данные в файле `.env`:
   - `DISCORD_TOKEN` — токен вашего Discord-бота.
   - `DISCORD_GUILD_ID` — ID вашего Discord-сервера (чтобы скопировать, включите режим разработчика в настройках Discord, нажмите правой кнопкой по серверу -> Копировать ID).
   - `LIST_FILE` — название файла со списком участников (по умолчанию `list.txt`).

## Запуск (Node.js) — Рекомендуемый способ

### Требования
- Установленный [Node.js](https://nodejs.org/) (версии 16.9.0 или выше).

### Инструкция по запуску
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Создайте файл списка (например, `list.txt`) и заполните его строками в формате:
   ```text
   Желаемый никнейм айди участника
   csero_copiya 1488228675242
   ```
3. Запустите скрипт:
   ```bash
   node rename_users.js
   ```

---

## Запуск (Python)

### Требования
- Установленный Python 3.8 или выше.

### Инструкция по запуску
1. Установите необходимые библиотеки:
   ```bash
   pip install discord.py python-dotenv
   ```
2. Подготовьте файл списка и запустите скрипт:
   ```bash
   python rename_users.py
   ```
